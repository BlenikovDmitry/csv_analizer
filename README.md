
### Описание:

Реализация анализатора произвольного csv файла с помощью библиотеки streamlit.
Все алгоритмы вынесены в функции с целью повторного использования.
Для удобства разработчика отказался от встроенных методов отрисовки графиков от streamlit, воспользовался возможностями Matplotlib

### Установка зависимостей:

Для работы программы нужны следующие модули:

- streamlit - отрисовка интерфейса;
- pandas - обработка данных датасета;
- matplotlib.pyplot - отрисовка графика;
- io - для скачивания графиков с сервера на клиентские станции;
- csv - отлавливаем ошибки парсинга - битые файлы;

### Запуск:
Запускаем командой streamlit run <путь к файлу app.py>

### Реализованный функционал:

- загрузка CSV-файл для анализа;
- отображение содержимого файла в виде таблицы;
- поля для выбора столбцов столбцов таблицы;
- статистический анализ выбранного пользователем столбца: среднее значение, медиана,среднеквадратичное отклонение);
- кнопка для загрузки построенного графика;
- построение статистических графиков для выбранного пользователем столбца: 
  график распределения в виде столбчатой диаграммы
- построение графика для выбранных пользователем пар столбцов;
- построение разных типов графиков: графики линии, диаграмма рассеяния (пример);
- проверка на битый файл;
- корректная обработка всех типов столбцов (числовые, текстовые, дата и время);
- автоматическая фильтрация нечисловых столбцов в выпадающих списках для статистики;
- автоматическая фильтрация нечисловых столбцов в выпадающих списках для числовых графиков;
- использование состояния сеанса и кеширования;

### Файлы:

- config.toml - настройки для интерфейса(цвета, шрифты). Важно! чтобы заработало, нужно положить в папку .streamlit в папке с программой;
- app.py - основной файл приложения. Запускать его командой streamlit run app.py
- bonds.csv - тестовый датасет - кусок реальной выгрузки результатов торговли ОФЗ с сайта ММВБ
