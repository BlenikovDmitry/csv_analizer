

Реализация анализатора произвольного csv файла с помощью библиотеки streamlit

Что умеет:

- загрузка CSV-файл для анализа;
- отображение содержимого файла в виде таблицы;
- поля для выбора столбцов столбцов таблицы;
- статистический анализ выбранного пользователем столбца: среднее значение, медиана,среднеквадратичное отклонение);



  Файлы:

  
  config.toml - настройки для интерфейса(цвета, шрифты). Важно! чтобы заработало, нужно положить в папку .streamlit в папке с программой. 
  
  app.py - основной файл приложения. Запускать его командой streamlit run app.py
